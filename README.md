# Image Classification using CNN (Deep Learning)

This project implements a Convolutional Neural Network (CNN) using TensorFlow/Keras to classify images from the CIFAR-10 dataset. The goal is to build a deep learning model that can accurately identify images across 10 categories such as airplanes, cars, birds, cats, dogs, and more.
